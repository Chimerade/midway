# Midway 1942 site (`web/`)

React + TypeScript application (Vite, React Router) for the
[Midway 1942](../README.md) project. It reads JSON files exported from the
SQLite database (`public/data/`, produced by `../data/export_data.py`) — no data
is hard-coded into the source. The UI is bilingual (English by default, French
toggle).

## Commands

```bash
npm install        # first time
npm run dev        # development server (http://localhost:5173/)
npm run build      # static build → dist/ (used by ../tout_regenerer.sh)
npm test           # Vitest tests
npm run lint       # ESLint
```

Data is normally regenerated by the root pipeline `../tout_regenerer.sh`. To
(re)build only the JSON export:

```bash
python3 ../data/export_data.py ../midway.sqlite public/data
```

## Structure

```
src/
├── main.tsx, App.tsx        # router + mount
├── layout/                  # menu + shell
├── theme/                   # light/dark theme context
├── i18n/                    # language context, strings, date formatting
├── routes/                  # Home, Map, Chronology, Methodology
├── components/ReplayMap/     # canvas engine (render.ts), ReplayMap, Controls, Legend, Feed
├── data/                    # JSON loading hooks
└── types/                   # TypeScript data contract
```

The drawing engine (`components/ReplayMap/render.ts`) is an iso-functional port
of the legacy single-file map `../carte_midway.html`.
